# 勉強時間管理アプリ

## 概要
勉強時間を記録・管理し、可視化することで学習の継続とモチベーション維持をサポートするデスクトップアプリです。

## 作成背景
-日々の勉強時間を把握したい
-学習のモチベーションを維持したい
-複雑な操作を避け、直感的に使えるツールを作りたかった。

##使用技術
- Python
-customtkinter(GUI構築)
-SQLite(データ保存)
-matplotlib(学習時間のグラフ表示)
-Git/Github(バージョン管理と公開)

##主な機能
-数字と科目を入力して、簡単に勉強時間を記録
-日別・科目別の学習時間をグラフで可視化
-データの保存・読み込みによる長期管理

##工夫した点
-操作の簡便さ、数字と科目を入力してボタンを押すだけの直感的操作
-モチベーション維持　: グラフ表示で達成感を可視化
-データ管理 : SQLiteでデータを永続保存し、後で振り返れる

## 苦労した点と解決方法
-GUIでの入力フォーム作成やレイアウト調整に時間がかかった。
　　　→CustomTkinterの公式ドキュメントとサンプルを参考に改善
- グラフ表示時にデータ形式のエラーが発生
  →データ型を統一し、入力チェックを追加


## 実行方法
pip install -r requirements.txt
python main.py
